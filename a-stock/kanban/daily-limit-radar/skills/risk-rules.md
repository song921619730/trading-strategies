# DLR 涨停雷达 — 风险过滤规则

> 策略版本：1.0.0
> 适用 Profile：risk_manager、analyst（主控）

## 硬淘汰条件（任一触发即排除）

| 编号 | 条件 | 说明 |
|------|------|------|
| R1 | 近 5 日涨幅 > 40% | 高位接盘风险极大 |
| R2 | 近期有大额减持/解禁公告 | 股东跑路信号 |
| R3 | ST、*ST、退市整理期 | 直接排除 |
| R4 | 连续 3 日净流出 | 资金持续撤离 |
| R5 | 业绩大幅预亏或亏损扩大 | 基本面恶化 |
| R6 | 所属板块整体退潮 | 板块龙头已见顶 |
| R7 | 监管问询/立案调查 | 政策风险 |

## 风险扣分规则

当标的通过硬淘汰后，按以下风险等级扣分：

| 风险等级 | 扣分 | 描述 |
|----------|------|------|
| 🔴 高危 | -5 分 | 多项风险叠加 |
| 🟠 中危 | -3 分 | 单一明显风险 |
| 🟡 低危 | -1 分 | 轻微瑕疵 |
| 🟢 安全 | 0 分 | 无明显风险 |

## 板块退潮判断标准

1. 板块龙头（最高板）出现放量阴线
2. 板块涨停数从 8+ 降至 3 以下
3. 板块内跟风股批量炸板
4. 龙头股出现监管关注函

## 高位惩罚规则

当标的满足以下 **全部** 条件时，强制降一档：
- 近 5 日涨幅 > 25%
- 连板数 ≥ 3
- 所属板块涨停数 ≤ 2（板块支撑弱）
- fc_ratio < 10（封单不坚决）

降级：`条件成立可执行` → `观察`
