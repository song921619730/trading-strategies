# DLR 涨停雷达 — 10 因子打分引擎

> 策略版本：1.0.0
> 数据来源：Tushare DB (ClickHouse)
> 适用 Profile：analyst（主控评分）、risk_manager（风险过滤）

## 10 因子评分体系（0-5 分制）

每个因子 0-5 分，满分 50 分。根据原版 `limit_line_radar.py` 逻辑定义。

### F1: 涨停强度 (0-5 分)
| 得分 | 条件 |
|------|------|
| 5 | 一字涨停，fc_ratio > 50，首次封板 < 09:35 |
| 4 | T 字涨停或快速封板，fc_ratio > 30，首次封板 < 10:00 |
| 3 | 常规涨停，fc_ratio > 15，首次封板 < 14:00 |
| 2 | 尾盘涨停或反复开板，fc_ratio > 5 |
| 1 | 勉强封住，fc_ratio < 5 |
| 0 | 炸板或数据异常 |

### F2: 连板高度 (0-5 分)
| 得分 | 条件 |
|------|------|
| 5 | 5 板及以上（龙头） |
| 4 | 4 板（准龙头） |
| 3 | 3 板（梯队核心） |
| 2 | 2 板（跟风加速） |
| 1 | 首板（新题材启动） |
| 0 | 无连板数据 |

### F3: 板块效应 (0-5 分)
| 得分 | 条件 |
|------|------|
| 5 | 所属板块涨停 ≥ 8 只（绝对主线） |
| 4 | 板块涨停 5-7 只（强主线） |
| 3 | 板块涨停 3-4 只（有板块支撑） |
| 2 | 板块涨停 1-2 只（弱板块） |
| 1 | 个股独立行情 |
| 0 | 板块数据缺失 |

### F4: 资金流向 (0-5 分)
| 得分 | 条件 |
|------|------|
| 5 | 超大单净流入 > 5 亿，连续 3 日净流入 |
| 4 | 超大单净流入 2-5 亿，连续 2 日净流入 |
| 3 | 超大单净流入 1-2 亿 |
| 2 | 超大单净流入 0.5-1 亿 |
| 1 | 超大单净流入 < 0.5 亿或净流出 |
| 0 | 资金流向数据缺失 |

### F5: 换手率匹配 (0-5 分)
| 得分 | 条件 |
|------|------|
| 5 | 换手率 5-15%（健康换手，充分洗盘） |
| 4 | 换手率 3-5% 或 15-25%（偏低或偏高但仍可控） |
| 3 | 换手率 1-3% 或 25-40% |
| 2 | 换手率 < 1% 或 > 40%（极端情况） |
| 1 | 一字板无量换手 |
| 0 | 换手率数据缺失 |

### F6: 市值弹性 (0-5 分)
| 得分 | 条件 |
|------|------|
| 5 | 流通市值 20-80 亿（最佳弹性区间） |
| 4 | 流通市值 10-20 亿或 80-150 亿 |
| 3 | 流通市值 5-10 亿或 150-300 亿 |
| 2 | 流通市值 < 5 亿或 > 300 亿 |
| 1 | 超大盘或微盘 |
| 0 | 市值数据缺失 |

### F7: 位置评估 (0-5 分) — 需要 120 日数据（半年）
> 数据来源：`tushare.tushare_stock_daily FINAL`（近 120 日 OHLCV）

| 得分 | 条件 |
|------|------|
| 5 | 股价 > MA20 > MA60 > MA120（完美多排），突破 120 日新高，底部启动 |
| 4 | 股价 > MA20 > MA60（多排），接近 60/120 日新高，上升中继 |
| 3 | 股价在 MA20 附近，距 60 日新高 10-20% |
| 2 | 股价在 MA20 下方，距 60 日新高 20-30%（已有一定涨幅） |
| 1 | 股价远离均线，距 120 日新高 > 30%（高位风险） |
| 0 | 历史数据不足 120 日 |

### F8: 成交量配合 (0-5 分)
> 数据来源：`tushare.tushare_stock_daily FINAL` + `tushare.tushare_daily_basic FINAL`
> 需要 60 日均量对比

| 得分 | 条件 |
|------|------|
| 5 | 量比 > 3，5 日均量 > 20 日均量，放量涨停 |
| 4 | 量比 2-3，5 日均量 > 20 日均量，温和放量 |
| 3 | 量比 1-2，均量持平 |
| 2 | 量比 0.5-1，缩量（需注意是否为一致加速） |
| 1 | 量比 < 0.5，极度缩量 |
| 0 | 量能数据缺失 |

### F9: 预期差/催化 (0-5 分) — LLM 评分
| 得分 | 条件 |
|------|------|
| 5 | 重大政策/事件催化，市场未充分定价 |
| 4 | 明确利好，有持续发酵空间 |
| 3 | 一般利好，已部分反映 |
| 2 | 利好出尽或催化不明确 |
| 1 | 无明显催化 |
| 0 | 存在负面催化 |

### F10: 技术共振 (0-5 分) — LLM 评分
> 数据来源：`tushare.tushare_stock_daily FINAL`（近 120 日 OHLCV，半年数据）

| 得分 | 条件 |
|------|------|
| 5 | 均线多排（MA5>MA20>MA60>MA120）+ MACD 金叉 + 突破 120 日关键位 |
| 4 | 2 个技术信号共振（均线 + MACD/突破/量价） |
| 3 | 1 个明确技术信号 |
| 2 | 技术信号模糊 |
| 1 | 技术面偏空 |
| 0 | 技术数据不足 120 日 |

## 三层分层收敛阈值

| 层级 | 阈值 | 状态标签 | 操作建议 |
|------|------|---------|---------|
| 待涨观察池 | ≥ 34 分 | `优先跟踪` | 等待突破信号 |
| 预备突破池 | 30-33 分 | `条件成立可执行` | 竞价/盘中确认 |
| 强确认跟踪池 | 26-29 分 | `观察` | 持续关注 |
| 淘汰 | < 26 分 | `暂不做` | 放弃 |

## 待涨优先原则

在最终候选中（建议 7 只），**待涨观察池 ≥ 50%**（至少 4 只）。
先找待涨，再找可执行，最后填观察。

## Tie-Breaker 规则（同分时）

1. 涨停时间更早者优先
2. fc_ratio（封单/成交比）更高者优先
3. 流通市值更小者优先（弹性更大）
4. 所属板块涨停数更多者优先
