# 📋 Proposal: 主升浪潜伏 — "温和突破"策略

**基于 V20/V21/V22 三轮实证 (2026-05-09)**

---

## 策略概要

| 特征 | 描述 |
|------|------|
| **策略名** | 主升浪潜伏 — 温和突破版 |
| **核心逻辑** | 均线粘合 + 低波动率 + 温和放量突破 + 牛市环境 |
| **信号频率** | ~1,100-2,300 次/年 |
| **数据范围** | 2020-2026 全市场 5,570+ 股票 |

---

## 入场信号

```
必须同时满足:
1. MA_CV_10d < 0.015          # 均线粘合度(MA5/10/20/60的10日平均变异系数)
2. ATR%(14) < 0.025            # 波动率低位
3. 1.5 < 量比 < 3.0            # 温和放量 (V22发现: >5 反而差)
4. 2% < 涨幅 < 5%              # 温和上涨 (V22发现: >7% 反而差)
5. 收盘价 > MA60               # 站上中长期均线
6. PE_TTM > 0                  # 排除亏损股
7. 总市值 ≥ 30亿               # 排除微盘股
8. 换手率 ≤ 全市场中位数        # V22发现: 高换手组显著劣于低换手组

环境过滤:
9. 沪深300 > MA60              # 仅在牛市环境下交易 (V21确认最优)
10. 优先中小盘 (<96亿)          # V22发现: 牛市+中小盘收益+2.38%
```

---

## 风控规则

| 规则 | 参数 | 依据 |
|------|------|------|
| 单信号仓位 | ≤ 2% (等权分散) | V21: 硬止损有害, 改用仓位控制 |
| 硬止损 | ❌ 不使用 | V21: -8%止损使均值从+1.42%暴跌至-3.58% |
| 动态止盈 | 回撤50%最高收益时平仓 | 保护右偏策略中的大赢家 |
| 行业分散 | 单行业敞口 ≤ 15% | 防止板块集中风险 |

## 预期表现

| 指标 | 基准(全市场) | V21(牛市) | V22优化(牛市+中小盘) |
|------|-------------|-----------|---------------------|
| T+20 均收益 | +1.53% | +2.11% | **+2.38%** |
| T+20 胜率 | 48.3% | 51.1% | **52.6%** |
| T+20 中位收益 | -0.26% | +0.26% | ~+0.30% |
| 年化信号数 | ~3,300 | ~2,300 | ~1,100 |

---

## 关键实证支撑

1. **V20**: 首次验证基础信号有效 (年化超额 16.51%)
2. **V21**: 
   - 洗盘信号无效 (p=0.158)
   - 硬止损有害 (51.7%触发率, 均值变负)
   - 牛市过滤最优 (年化超额 14.03%)
3. **V22**:
   - 温和突破优于强力突破 (涨幅2-5% + 量比1.5-2 最优)
   - 低换手优于高换手 (Q1组 +2.07% vs Q4组 +0.25%)
   - 小盘优于大盘 (牛市+中小盘 +2.38%)
   - 多重增强无效 (组合过滤反而劣于单因子)

---

## 建议行动

1. **短期**: 将"温和突破"规则整合入短线策略选股增强
2. **中期**: 在短线策略中增加牛市环境开关(沪深300>MA60)
3. **长期**: 结合资金流向因子(moneyflow)进一步验证
