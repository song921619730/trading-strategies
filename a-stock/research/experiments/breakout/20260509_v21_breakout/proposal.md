# 📜 Proposal: 主升浪潜伏信号 — 牛市过滤增强规则

## 🚨 Problem Statement

V20 已验证"均线粘合 + ATR低位 + 放量突破"信号的Alpha能力 (年化超额 12.4-16.5%)，但存在三个问题:
1. **胜率偏低** (47.9%)，中位收益为负 (-0.34%)
2. **熊市大幅衰减** (熊市仅 +0.17% 均收益，41.6% 胜率)
3. **右偏分布** (偏度 4.03)，尾部风险大 (最大亏损 -50%)

V21 进一步测试了"洗盘信号"和"-8%止损"，结果均为负面:
- 洗盘信号不显著 (p=0.158)，反而拉低收益
- -8%硬止损摧毁Alpha (均值从 +1.42% → -3.58%)

**需解决**: 如何在不破坏右尾Alpha的前提下，提升胜率和稳健性？

## 💡 Proposed Rule

### 牛市环境过滤 (Primary)

```python
# 市场环境判断: 沪深300指数收盘价 > MA60
csi300_close = get_index_close('000300.SH')
csi300_ma60 = csi300_close.rolling(60).mean()
bull_market = csi300_close > csi300_ma60

# 仅在市场为牛市时启用主升浪信号
if bull_market:
    score += breakout_signal_score(stock)
```

### 信号排序加分 (Secondary)

对满足以下条件的股票给予排序加分:
- MA_CV_10d < 0.02 (均线粘合)
- ATR(14)/Close < 0.025 (波动率收缩)
- 量比 > 1.5 且 涨幅 > 2% 且 Close > MA60 (放量突破)
- PE_TTM > 0 且 总市值 ≥ 30亿

### 风控规则

- **不使用硬止损**: 实证证明 -8% 硬止损摧毁 Alpha
- **仓位控制**: 单信号仓位 ≤ 2% (等权分散 50 只)
- **行业限制**: 单行业敞口 ≤ 15%
- **动态跟踪止盈** (可选): 突破后回撤 50% 最高收益时平仓

## 📝 Reviewer Notes

### 关键发现

1. **牛市过滤是最佳增强**: 年化超额提升 1.63%，胜率提升 2.7%，中位收益从负转正
2. **洗盘信号不可用**: 仅 351 个样本 (1.56%)，p=0.158 不显著，均收益反而更低
3. **硬止损不可用**: 51.7% 触发率，均值从 +1.42% 暴跌至 -3.58%
4. **策略本质**: 右偏尾部驱动型 — 大部分信号小亏，少数大赢家贡献Alpha。适合做多分散持仓，不适合集中+止损

### 风险提示

- 策略胜率始终 < 52%，不适合追求高胜率的交易者
- 信号在牛市中有效，但牛市判断本身有滞后性 (MA60 是滞后指标)
- 建议在实盘前进行至少 3 个月的模拟盘验证
