## 🎯 Research Question

**核心假设**: "均线粘合 + ATR 低位 + 放量突破" 组合在 20 日内跑赢沪深300指数年化超额 15% 以上。

**逻辑链**:
1. 均线粘合 (MA CV < 2%) → 多空力量均衡，方向选择在即
2. ATR 低位 (ATR% < 2.5%) → 波动率收缩 (VCP 形态)，突破后往往伴随大幅波动
3. 放量突破 (量比>1.5, 涨>2%, 站上MA60) → 资金进场确认，趋势启动信号

## 📐 Methodology Details

### 信号定义
```
Signal = (MA_CV_10d < 0.02)          # 近10日均线粘合
       AND (ATR_14/Close < 0.025)    # 波动率低位
       AND (Vol_Ratio > 1.5)         # 放量
       AND (Pct_Chg > 2%)            # 涨幅确认
       AND (Close > MA60)            # 站上长期均线
       AND (PE_TTM > 0)              # 盈利
       AND (Total_MV >= 30亿)        # 市值门槛
```

### 统计方法
- **前向收益**: 信号触发后持有 5/10/20 日的收益率
- **超额收益**: 个股收益 - 同期沪深300指数收益
- **市场环境**: 以沪深300收盘价 vs MA60 划分牛/熊市
- **样本量**: 25,871 个信号，覆盖 2020-2026 全周期

### 计算引擎
- 因子计算 (MA, ATR, CV) → ClickHouse 窗口函数 (服务端)
- 前向收益 → pandas 分组 shift 计算
- 超额收益 → 与沪深300逐日对齐

## 💡 Conclusion

### 假设验证结果: **部分成立** ✅

1. **年化超额 16.51%** > 15% 阈值 — **假设在统计意义上成立**
2. 但策略呈现 **高度右偏分布** (偏度 6.19): 中位收益为负 (-0.4%)，大部分信号亏损，少数大赢家拉高均值
3. **胜率仅 47.7%** < 50%，不适合追求高胜率的短线交易者
4. **牛市显著优于熊市**: 牛市超额 1.50%，熊市仅 0.86%
5. **2022年熊市表现最差**: -2.38% 收益，29.1% 胜率 — 需要市场过滤

### 策略特征画像

| 特征 | 评估 |
|------|------|
| Alpha 能力 | ✅ 有 (年化超额 16.5%) |
| 胜率 | ⚠️ 偏低 (47.7%) |
| 盈亏比 | ✅ 良好 (1.66) |
| 分布形态 | ⚠️ 高度右偏，尾部驱动 |
| 市场依赖 | ⚠️ 牛市有效，熊市衰减 |
| 信号频率 | ✅ 充足 (年均 ~4,000 次) |

### 关键发现

1. **这不是高胜率策略，而是"尾部收益"策略**: 大部分信号小亏，少数信号大赚。适合作为选股增强因子而非独立交易系统。
2. **均线粘合度是核心**: MA CV < 2% 筛选出的是"蓄势待发"的整理末期股票，一旦突破往往爆发力强。
3. **环境过滤至关重要**: 牛市中策略表现优异 (2.30%, 50.3%胜率)，熊市大幅衰减。建议加入指数趋势过滤 (沪深300 > MA60 时才启用)。
4. **参数敏感性**: 收紧条件 (CV<0.01, ATR<0.015) 信号过少; 放宽条件 (CV<0.03, ATR<0.035) 样本增至 85,798 但平均收益可能下降 (需进一步验证)。

### 后续优化方向

1. **加入止损**: 由于偏度高，单笔最大亏损可达 -50%，需引入止损规则 (如 -8% 或 -ATR×3)
2. **板块共振过滤**: 结合新闻Pipeline，仅在板块有催化剂时介入
3. **仓位管理**: 按信号强度 (量比、突破幅度) 分级持仓
4. **持有期优化**: 测试 10日 vs 20日 vs 动态止盈 (突破后回撤止盈)

## 📝 Proposal

基于本次实证，建议将此信号作为 **选股增强因子** 整合入短线策略:

1. **过滤规则**: 排除 MA_CV_10d > 80分位的股票 (即均线发散的股票)
2. **加分规则**: 对同时满足 ATR低位 + 放量突破 的股票给予排序加分
3. **环境限制**: 仅在沪深300处于 MA60 上方时启用该因子
4. **需配合风控**: 单笔止损 -8%，组合层面限制单行业敞口
