# 📜 提案: 量价背离 + 机构资金流 选股增强规则

**Status**: 🟡 Draft  
**Linked Experiment**: `20260509_v14_auto`  
**Target Strategy**: `kanban/screening/kanban/strategy/a-stock-shortline`

## 🚨 Problem Statement

当前短线策略在选股排序中主要依赖涨停板相关因子(连板数、封板时间、开板次数等), 但缺少对**量价关系**和**资金流向**的考量。本研究验证了两个独立有效的增强因子:

1. **量价背离因子**: 价格创新高但缩量的股票, 次日收益显著更低 (-11.1bp, p<0.001)
2. **机构资金流因子**: 大单+超大单净买入占比高的股票, 次日收益显著更高 (+12.0bp, p<0.001)

## 💡 Proposed Rule

### 规则A: 量价背离过滤 (防御性)
```
计算: vp_div = (close / high_10) - (vol / vol_ma_10)
IF vp_div > 80th_percentile THEN 降权或排除
```
**逻辑**: 缩量创新高的股票上涨动能衰竭, 次日大概率走弱

### 规则B: 机构资金流加权 (进攻性)
```
计算: inst_ratio = (超大单净 + 大单净) / 总成交额
按 inst_ratio 分5组, Q5(最高)组给予额外排序加分
```
**逻辑**: 机构资金持续流入的股票次日有正向溢价

### 组合规则
```
最终得分 = 基础得分 + inst_rank_bonus - vp_div_penalty
WHERE:
  inst_rank_bonus = 2.0 (if inst_ratio >= 80th_pct) else 0
  vp_div_penalty  = 1.5 (if vp_div >= 80th_pct) else 0
```
