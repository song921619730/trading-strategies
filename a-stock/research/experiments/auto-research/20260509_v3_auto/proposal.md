# 📜 提案：涨停资金流结构过滤器

**状态**: 🟡 草稿  
**关联实验**: `20260509_v3_auto`  
**目标策略**: `a-stock-shortline`

## 🚨 问题陈述

当前 `a-stock-shortline` 策略在涨停候选股筛选中主要依赖价格行为（连板数、开板次数）和技术面指标，未纳入资金流向结构信息。实验 `20260509_v3_auto` 发现：

1. 超大单净买入占比与次日开盘溢价呈中等正相关（r = 0.33）
2. 大单净买入占比最高组(Q5)的开盘溢价(3.52%)是最低组(Q2)的 2.7 倍
3. 低换手(<8%) + 高机构参与是最强开盘信号组合（高开>3%概率 46.7%）
4. 大单净卖出的涨停股次日开盘表现显著偏弱

**缺失**：缺乏对涨停日"谁在买"的判断依据。

## 💡 建议规则

在现有筛选流程的涨停候选池之后，增加资金流结构过滤器：

```
IF 涨停日超大单净买入占比(elg_ratio) > 0.10:
    → 加分 +2

IF 涨停日换手率 < 8%:
    → 加分 +1

IF 涨停日大单净卖出(large_net_ratio < 0):
    → 减分 -2

IF 涨停日开板次数 >= 4:
    → 减分 -1

最终评分 >= 2: 进入高优先级候选池
最终评分 <= -1: 从候选池剔除
```
