# 📜 Proposal: GSR动量配对信号 (Gold-Silver Ratio Momentum)

**Status**: 🟡 Draft  
**Linked Experiment**: `20260509_v8_auto`  
**Target Strategy**: `single-agent/pure-ai-cio`

## 🚨 Problem Statement

pure-ai-cio 策略在贵金属品种选择（黄金 vs 白银）上缺乏量化信号。当前市场环境（高油价+美联储鹰派+地缘冲突）下，白银因工业属性面临双重压力，但策略无法系统性识别这种分化。

历史回测表明：
- 复合鹰派 Regime 内，白银年化收益**落后黄金 63.85%**
- 金银比(GSR)20日变化与白银未来20日相对收益的**相关系数达 -0.76**
- 五分位分组呈现完美单调梯度，极差 12.06% (p<0.001)

## 💡 Proposed Rule

### 规则 1: GSR动量方向信号

```
GSR = XAUUSDm.Close / XAGUSDm.Close
GSR_20d = (GSR - GSR.shift(20)) / GSR.shift(20)

IF GSR_20d > +0.08:
    → 信号: 白银相对弱势
    → 行动: 优先黄金多头，避免白银多头
    → 信心: 白银未来20日相对收益预期 -6.13%

IF GSR_20d < -0.08:
    → 信号: 白银均值回归
    → 行动: 优先白银多头，关注白银突破
    → 信心: 白银未来20日相对收益预期 +5.93%

IF |GSR_20d| < 0.08:
    → 信号: 中性
    → 行动: 按原有逻辑选择品种
```

### 规则 2: 鹰派 Regime 贵金属配置倾斜

```
IF (UKOILm.Close > 95) AND (USDCHFm 20日收益率 > 0):
    → 激活"鹰派贵金属预警"
    → 贵金属仓位: 黄金 ≥ 75%, 白银 ≤ 25%
    → 白银多头止损: ATR × 1.0 (从默认收紧)
```

## 📝 Reviewer Notes

*本研究基于 2018-2026 共约 8 年的白银-黄金数据。GSR 动量信号的统计显著性极高 (p<0.001)，但需要注意：*
- *2022年俄乌冲突占 Regime 天数的 83%，结果可能受单一事件驱动*
- *2025年白银大幅跑赢黄金 (+36.1%) 表明鹰派信号解除后白银均值回归动力强劲*
- *建议先在影子模式中运行，积累足够信号后再升级为正式规则*
