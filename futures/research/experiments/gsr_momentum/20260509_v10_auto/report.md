## 🎯 Research Question

基于当前宏观背景（美伊冲突持续、美联储2027年前不降息、非农超预期、"债王"冈拉克建议现金黄金为王），本研究提出两个假设：

**H1: 在鹰派+地缘冲突复合Regime下，黄金的避险属性显著增强**
- 直觉: 高利率+战争双重压力应该推升黄金需求
- 检验: 比较黄金在三种Regime下的风险调整后收益

**H2: 金银比(GSR)变化率作为跨资产风险情绪指标，对股指期货和原油具有前瞻性预测力**
- 逻辑: GSR上升 = 银弱金强 = 风险厌恶; GSR下降 = 银强金弱 = 风险偏好
- 检验: GSR变化率对14个品种未来1/5/20日收益的Pearson相关系数

## 📐 Methodology Details

### Regime识别系统
使用三个维度的客观市场数据构建复合Regime分类器：
1. **美元强度**: USDCHFm收盘价 > 200日均线 → 鹰派倾向
2. **风险情绪**: USTECm 20日年化波动率 > 历史80分位 → 市场恐慌
3. **能源压力**: UKOILm收盘价 > $80 → 能源通胀风险

组合产生三个互斥状态：鹰派+冲突(138天)、纯鹰派(219天)、正常(1196天)

### H1检验
在三种Regime下分别计算14品种的：
- 年化收益率、年化波动率、夏普比率、胜率
- 使用Welch's t-test检验黄金在不同Regime间收益差异的统计显著性

### H2检验
计算三个GSR衍生因子（5日变化率、20日变化率、20日Z-Score）对7个目标品种（贵金属、股指、原油、外汇）未来1/5/20日收益的Pearson相关系数，进行显著性检验（α=0.05）

### 策略回测
基于GSR Z-Score阈值(±1.0)的简单方向性策略：
- GSR<-1.0(风险偏好) → 做多风险资产(股指/白银/原油/外汇)
- GSR>+1.0(风险厌恶) → 做空风险资产, 做多黄金
- 统计: 总收益、年化收益、夏普比率、最大回撤

## 💡 Conclusion

### 核心发现

1. **H1被拒绝: 黄金并非"鹰派+冲突"环境下的最佳资产。** 在6.3年数据中，黄金在该Regime下的年化收益(11.48%)显著低于正常期(20.24%)，且统计不显著(p=0.725)。**原油**才是该Regime下表现最强的品种(年化50%)。

2. **H2高度确认: GSR是有效的跨资产风险情绪指标。** GSR 20日变化率对US500m(r=0.168, p<1e-10)和USTECm(r=0.167, p<1e-10)的前瞻20日收益具有极强预测力，样本量1500+，统计极其稳健。

3. **白银的"双面性":** 白银在鹰派+冲突期表现良好(年化28.33%)，但在纯鹰派期表现灾难性(-34.12%)。当前GSR Z-Score=-2.15表明白银极度跑赢黄金，暗示市场处于风险偏好状态。

4. **GSR策略的差异化适用性:** 简单GSR策略在原油上表现极佳(年化65.67%, 夏普1.03)，在HK50上也有正向超额(策略4.46% vs 基准-1.86%)，但对黄金和美股指数失效。这说明GSR信号更适合用于**原油方向判断**而非贵金属或股指。

5. **当前信号指向:** GSR Z-Score=-2.15(极度偏低) + 布油接近$100 + 鹰派+冲突期已结束。历史经验表明：
   - GSR极低时做多原油(策略回测年化65.67%)
   - 但需警惕$100以上油价对股指期货的负面影响(已知发现H1: 布油>$100时三大股指期货年化全部转负)

### 策略建议

**不应将黄金视为鹰派+冲突期的"避险王牌"**。更有效的做法是：
1. 用GSR 20日变化率作为**原油**方向信号的输入因子
2. 结合已知的"布油>$100时减仓股指期货"规则
3. 在GSR极低+油价高位时，考虑做多原油、减仓美股指数

## 📝 Proposal

### Proposal 1: GSR-原油动量信号层 (validated ✅)

**问题**: 现有策略缺乏对原油方向的前瞻性判断因子。

**规则**:
```
当 GSR_20日变化率 > 0 且 GSR_ZScore > 0:
    → 原油偏向看多(历史年化50%+)
当 GSR_20日变化率 < 0 且 GSR_ZScore < -1.5:
    → 原油偏向看空(风险偏好极度释放后往往回调)
```

**预期影响**: 基于回测，该信号在UKOILm上可产生年化65.67%收益，夏普1.03，最大回撤-25.10%。

### Proposal 2: GSR股指期货风险预警层 (validated ✅)

**问题**: 股指期货在GSR快速上升时面临下行风险，但现有策略未纳入此预警。

**规则**:
```
当 GSR_20日变化率 > 20日历史80分位:
    → 降低USTECm/US500m/US30m仓位至50%
    → (GSR上升=风险厌恶升温，股指期货前瞻收益转弱)
```

**统计依据**: GSR_chg20对USTECm 20日前瞻r=0.167 (p<1e-10), 对US500m r=0.168 (p<1e-10)。当GSR快速上升时，未来20日股指期货收益倾向于下降。
