# Hermes Trading Strategies

This repository contains all trading strategies, categorized by market and architecture.

## Structure

- **futures/**
    - **single-agent/**: Lightweight, autonomous agents (e.g., `pure-ai-cio`).
    - **kanban/**: Multi-agent collaborative pipelines (e.g., `macro`).
- **a-stock/**
    - **kanban/**: A-share screening pipelines.
- **archive/**: Retired strategies.
